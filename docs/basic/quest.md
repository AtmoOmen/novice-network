##### 做任务常见问题

* 如果发现任务图标变成<img src="/images/icons/071013.png" class="no-zoom sm-icon" />，这通常是玩家角色不满足任务要求所致，右键任务NPC可以查看具体原因。
* 如果你迷路、跑丢了，不知道该去哪，请参考[地图与移动](/basic/map.md)章节。
* 有些任务需要对**指定地点**使用情感动作/道具，或者在**指定地点**等待。需要右键选中“指定地点”，并对着它使用情感动作/道具。
* 交纳任务道具时，可以直接在交纳窗口上<i class="xiv mouse-right-button"></i>点击右键，上面会自动弹出符合要求的物品（不用从背包里找到道具拖过去）。

## 职业/职能任务

在跟随主线剧情之余，你需要每5级回到职业行会，接新的职业任务。职业任务会在30级的时候转为特职任务，特职导师也不会停留在原本的职业行会里，如果迷路了，可以使用主线指引提示任务位置。

30级职业任务会送给玩家职业水晶，玩家装备上职业水晶之后，就算习得了特职（摘掉水晶则会恢复为普通职业）。之后的职业任务也会送给玩家职业专属套装和职业技能，满足职业任务等级条件的玩家，应该优先完成职业任务。

玩家在15级之后，世界各地的冒险者指导教官会出现<img src="/images/icons/061417.png" class="no-zoom sm-icon" />任务，这是初学者学堂的任务，无论你是否有MMORPG经验，都推荐你完成这个任务（需要先完成15级的职业任务）。初学者学堂任务会按T奶DPS职能区别，每个职能会有5~7个战斗指导任务，完成所有指导任务之后，会赠送职能适合的一身装备，和提升获取经验的饰品。非常适合新人练级。

## 重要任务

在游戏进程中，会遇到部分带有<img src="/images/icons/061419.png" class="no-zoom sm-icon" />标记的任务，它们都是重要任务，涉及到各种功能、副本的开放，如果你在主线过程中看到了这些任务，推荐尽早完成哦。

**※ 注意：名称带有“登峰造极”字样的任务为高难度副本（极神）的开启任务，由于副本本身难度较高，无意挑战的玩家可以不参与。想要挑战的玩家推荐先看过攻略后，通过招募板等方式召集队友进行挑战。**[点我了解FF14挑战副本的类型和相关介绍](/topic/battle.md)。

**※ 注意：<Quest name="传说中的武器工匠" type="plus" />/<quest name="复苏的上古武器" type="plus" />，<Quest name="其名元灵武器" type="plus" />/<Quest name="创造新武器" type="plus" />分别是50级和60级的[发光武器](/topic/shine.md)系列支线，属于需要消耗大量时间的游戏内容，请先参照相关介绍确认自己是否想要制作该武器，再决定是否进行这系列任务。**

### 必做重要任务

对于新人来说，下列任务会开放各种游戏功能，务必及时完成（直升包不包含这些任务，因此直升玩家也要去做）：

* 主线10级：<quest name="挑战理符" search type="plus" />、<quest name="行会特令" search type="plus" />，完成对应主城9级主线任务之后，在出生城冒险者行会接取。只有完成挑战理符任务之后，才能开放其他地区的理符。
* 主线15级：<quest name="遗失的挑战笔记" type="plus" />，解锁每周挑战笔记，完成笔记上的项目可以获得大量经验、金钱、以及金碟币。
* 主线17级：<quest name="雇员大探险" search type="plus" />，任意主城冒险者行会接取，完成后开放雇员探险功能。[雇员系统详解](#)。
* 主线19级：<quest name="情感培育之力" type="plus" />，把精炼度满的装备转化为魔晶石。[魔晶石系统](https://ff14.huijiwiki.com/wiki/%E9%AD%94%E6%99%B6%E7%9F%B3)。
  * <quest name="情感凝聚之力" type="plus" />，合成魔晶石。
* 主线20级：<quest name="我的专属陆行鸟" search type="plus" />，完成20级主线任务<quest name="前行之路" search type="main"/>并加入大国防联军后接取，完成任务可获得坐骑专属陆行鸟（**在获得专属陆行鸟之前，即便是商城购买的坐骑也无法使用**）。
  * 任意职业30级：<quest name="可靠的搭档" type="plus" />，解锁专属陆行鸟辅助战斗及陆行鸟鞍囊（扩展背包），在黑衣森林南部林区恬静路营地接取。
* 任意职业30级：解锁特职，务必及时完成30级职业任务，并获得灵魂水晶转为特职。
  * **60、70、80级职业任务**：完成后会解锁相应技能，务必及时完成。
* 30级：<quest name="狱中斗狼" search type="plus" />、<quest name="法外纷争" search type="plus" />，开放PVP竞技场和战场。
* 风脉任务：任务奖励有<img src="/images/icons/060033.png" class="no-zoom sm-icon" />标识的即为风脉任务，务必及时完成。

## 理符任务

理符任务现在并不是重要的任务系统，但是由于它以前占有很大的游戏比重，因此稍显复杂。在完成出生主城的<quest name="挑战理符" search type="plus" />任务之后，各地会出现==XX理符开放任务==，完成对应地区的理服开放任务之后，就可以接取当地的理符任务了。

理符任务分为行会理符和军队理符，所有理符任务又可以分为战斗、生产、采集三大类。目前只有低级（30级以下）的战斗职业，生产采集职业，以及需要制作发光武器的玩家，会需要完成理符任务。如果看到理符开放任务，记得及时完成。

部分理符是跨地区发行/递交的，则需要同时开启两个地区的理符才可以。

理符有理符连接功能，即小队中所有人接取了相同的理符，那么队伍中只要有1人开启理符任务，其他人的相同理符会同时开启，共享任务进度和完成状况。

::: collapse 理符任务的接取与完成

在开启对应地区理符功能后，就可以在理符发行人那接取理符任务。每名玩家都有受理限额，每12小时回复3个受理限额，上限为99个（放弃任务并不会退回受理限额）。
